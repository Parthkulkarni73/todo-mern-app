🚀 MERN Todo App

A full-stack Todo application built using the MERN stack with secure backend deployment and cloud hosting.

🔗 Live Demo

Frontend: https://vercel.com/parthkulkarni73s-projects/todo-mern-app


Backend API: https://todo-mern-backend-2474.onrender.com

🛠 Tech Stack

Frontend: React (Vite), Tailwind CSS

Backend: Node.js, Express.js

Database: MongoDB Atlas

Hosting: Render (Backend), Vercel (Frontend)

✨ Features

Add, Update, Delete Todos

RESTful API

Cloud Deployment

Environment-based API config

Production-ready structure

⚙️ Installation (Local Setup)
git clone https://github.com/Parthkulkarni73/todo-mern-app.git
cd backend
npm install
cd ../frontend
npm install
🌍 Deployment

Backend deployed on Render
Frontend deployed on Vercel
